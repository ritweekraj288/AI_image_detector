# RitweekAI

RitweekAI is a web-based platform that allows users to upload images and determine whether they are AI-generated or real.

## Live Demo
**Deployed Link:** https://ritweekai.vercel.app

## Tech Stack
- AI/ML: Custom AI model  
- Backend: API-based service  
- Frontend: Web interface  
- Deployment: Cloud platform
